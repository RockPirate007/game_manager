# Football Manager 2026

Текстовый футбольный менеджер на Python + C++ (pybind11).

## Запуск

```bash
make        # собрать C++ модуль
python main.py
```

## Архитектура

- `cpp/` — C++ ядро: движок матчей, ИИ клубов, калькулятор статистики
- `core/` — GameState, EventBus, константы
- `engine/` — Python-обёртки над C++
- `game/` — карьера, сезон, календарь, новости
- `models/` — модели данных: игрок, команда, матч, лига
- `services/` — трансферы, финансы, тренировки, молодёжь, контракты, сохранение
- `ui/` — rich-интерфейс: меню, виджеты, консоль
- `data/` — JSON: лиги (EPL, La Liga), клубы, игроки, шаблоны
- `utils/` — логгер, RNG, форматеры, валидаторы

## Требования

- Python 3.10+
- rich
- pybind11 (для C++ модуля)
- clang++ (опционально, для C++ ядра)
